MAC系统Github和Hexo快速搭建自己的博客

今天想搭建一个博客，来记录一下自己的工作、学习和生活，因为意识到自己在之前的工作和生活中缺少一些记录。这篇文章作为站点的第一篇，记录了我是如何基于Github和Hexo在MAC环境下搭建属于自己的博客。

使用技术简介:

> - Github: 作为开源代码库以及版本控制系统，Github目前拥有140多万开发者用户。随着越来越多的应用程序转移到了云上，Github已经成为了管理软件开发以及发现已有代码的首选方法。 - Github Pages: Github官方提供的免费的静态站点服务，三个特点：免费托管、自带主题、支持自制页面和Jekyll。 - Markdown: Markdown是一种轻量级的标记语言，由John Gruber和Aaron Swartz创建，使其成为可读性最大并可再发行的可输入输出的格式。这种语言创建灵感来自于已经存在的带标记的电子邮件文本。 - Hexo: 使用Node.js语言编写的一套博客系统。

步骤概要

> 1. 注册Github账号，创建项目 2. 本地安装Hexo环境 3. 创建你的博客并提交同步 4. 自定义博客
